# KRATOS UI Bible

## Propósito
KRATOS Mission Control é um cockpit local-first neurocompatível que responde rapidamente: onde estou, o que faço agora, o que está rodando, o que quebrou, onde parei e qual a próxima ação.

## O que KRATOS não é
- SaaS genérico
- ERP
- Grafana cru
- CRM tradicional
- dashboard bonito vazio

## O que KRATOS é
- mundo operacional vivo
- sistema operacional emocional
- mission control visual
- painel de retomada de contexto
- cockpit pessoal para operador solo

## Direção visual
- Apple-clean
- Raycast / Linear / Arc / Vercel como referência de disciplina
- HUD futurista limpo
- mundo vivo com ilhas
- pseudo-3D seguro
- game UI premium adulta
- sem excesso de neon
- sem fantasia infantil

## Hierarquia visual obrigatória
1. Missão atual
2. Próxima ação
3. Estado do sistema
4. Alertas relevantes
5. Foco do dia
6. Agenda
7. Squads / Aurora

## Regras neuro UX
- Uma ação primária por viewport
- Textos curtos e específicos
- Severidade clara
- Recuperação de contexto acima de exploração
- Evitar ornamentos que não carregam informação
