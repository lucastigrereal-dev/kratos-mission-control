# Requisitos de Screenshots

Para realizar a QA visual (microfase 07), colete os seguintes prints:

1. **Tela Inicial (Map)** — Capturar o mapa completo com ilhas, castelo, HUD superior, lateral e dock inferior visíveis.  Tamanho recomendado: 1920×1080.
2. **Painel Aurora** — Abrir e capturar o painel da Aurora com orb, cards e sugestões.
3. **Página OMNIS** — Capturar a página do Omnis Lab (mesmo que seja placeholder), incluindo a HUD.
4. **Página Akasha** — Capturar a página do Akasha Vault.
5. **Outro Island Pages** — Uma captura por cada página de ilha que foi implementada.
6. **Preferência de Motion** — Capturar a tela inicial com `prefers-reduced-motion` ativado para mostrar que as animações foram desativadas.

As imagens devem ser salvas em `reports/screenshots/` (fora deste pacote) para referência.  Não inclua as imagens no repositório final se elas forem grandes; armazene localmente ou no sistema de relatórios da equipe.
